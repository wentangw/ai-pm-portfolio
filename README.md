# ai-pm-portfolio
AI Product Manager Portfolio

This portfolio was developed using an AI-assisted workflow.

Tools used:

Cursor for AI-assisted coding

ChatGPT for prompt engineering

Vercel for deployment.

Development process:

MVP prototype → prompt-driven iteration → UX optimization → deployment.

该作品通过 AI 辅助开发流程完成。

使用工具：

Cursor（AI 编程助手）

ChatGPT（Prompt 设计）

Vercel（部署平台）

开发流程：

MVP 原型 → Prompt 驱动迭代 → UX 优化 → 产品部署。

- 在终端运行：
```
npx create-next-app@latest ai-pm-portfolio
cd ai-pm-portfolio
npm install tailwindcss framer-motion
npm run dev
```
- 然后打开：
 http://localhost:3000
- 就能看到我完成的关于AI产品经理作品集的展示网站。
